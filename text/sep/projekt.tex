%Abstrakt (cesky, anglicky)
\abstract[cs]{Tato práce se zabývá tvorbou obrazových filtrů pomocí souběžného učení v~koevolučním algoritmu. V~první části textu jsou popsány evoluční algoritmy, zejména genetický algoritmus a kartézské genetické programování. Dále je představena jejich praktická aplikace na úloze návrhu obrazových filtrů. Poté je uvedeno, jak lze zavedením koevoluce s~druhou populací urychlit hledání vhodného řešení. V~poslední teoretické části se text zabývá vztahem učení a evoluce, Baldwinovým efektem a plasticitou fitness a uvádí několik příkladů použití učení v~evolučních algoritmech. V~závěrečné kapitole je navržen systém pro tvorbu obrazových filtrů využívající souběžné učení v~koevolučním algoritmu, který bude implementován v~navazující diplomové práci.}

\abstract[en]{This project deals with image filter design using colearning in coevolutionary algorithm. First part of thesis describes evolutionary algorithms, mainly genetic algorithm and cartesian genetic programming, and their practical usage on the task of image filter design. Then it is shown how the search for the solution can be accelerated by introducing coevolution with second population. The following part deals with the relationship between learning and evolution, Baldwin effect and fitness plasticity and provides several examples how learning was used in evolutionary algorithms. The last chapter of the thesis describes the concept of the proposed algorithm that generates image filters using colearning in coevolutionary algorithm. The proposed algorithm will be implemented as a part of the master's thesis.}
